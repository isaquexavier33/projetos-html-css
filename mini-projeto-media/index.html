Testando Media Queries

Media Queries são úteis para quem quer tornar seu site responsivo, mas o que é responsividade? Falando de forma simples, responsividade é conseguir responder a diferentes tamanhos e resoluções de tela(celular, tablet, tv, desktop, laptop, etc).

Media Query = Media Type + Media Feature, mas o que é um Media Type?

Media Types

Os Media Types são para especificar o tipo de mídia que um arquivo ou uma regra CSS vai tratar, existem diversos deles, entre eles, os mais utilizados são: screen(para telas), print(para impressoras) e all(para todo tipo de mídia).

Ah tá, entendi. Mas o que são Media Features?

Media Features

As Media Features são as características de uma mídia, por exemplo, screen, é para telas, mas telas de quais tamanhos e de quais resoluções? Para isso usamos as media features max-width: e min-width:, digamos que eu quero que um site se adapte de acordo com a orientação da tela, usamos a media feature orientation: portrait(retrato) ou landscape(paisagem).
